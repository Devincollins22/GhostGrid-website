The Journey

From IT Support to Cloud Security

I grew up around CAC Computer Inc., my family's IT company that's served Western New York for 12 years — so troubleshooting and fixing real infrastructure isn't new to me. What's new is where I've pointed that experience: earning an M.S. in Cybersecurity at Canisius University, passing the ISC2 Certified in Cybersecurity (CC) exam, and now going deep on AWS security and DevSecOps — IAM, network configuration, secure pipelines, and the practical work of hardening cloud environments.

The long-term plan is straightforward: keep building real skills, keep taking on real AWS security work, and grow that into a company that manages cloud security for other businesses.
